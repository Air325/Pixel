# Pixel
Графический Редактор, позволяющий Вам рисовать и сохранять нарисованное! Среди инструментов доступно: 5 цветов, ластик, заливка.

<h3>Инструменты:</h3>
<ul>
  <li><a href="https://ru.wikipedia.org/wiki/HTML">HTML</a></li>
  <li><a href="https://ru.wikipedia.org/wiki/CSS">CSS</a></li>
  <li><a href="https://ru.wikipedia.org/wiki/JavaScript">JavaScript</a></li>
</ul>
